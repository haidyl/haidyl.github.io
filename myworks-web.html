Case Study 1: AtriCure

AtriCure is a healthcare company which deals with patients with Atrialfibrilation. I was responsible for the UI of the Website design. This is a live project https://atricure.com. I was involved with the AtriCure team from the scratch till launch of the website. I worked with the design team, used Figma as a designing tool for the layouts. Later, the UX design was translated into HTML layouts using latest Bootstrap Framework. Finally that was converted to Wordpress CMS Website (with collaborating with a WordPress expert)
